# PVE Manager 部署说明

## 🚀 快速部署到本地环境

### 环境要求
- Node.js 16+
- npm 8+
- 能访问PVE服务器的网络环境
安装命令：
curl -fsSL https://deb.nodesource.com/setup_18.x | bash -
apt install -y nodejs

### 1. 启动应用
```bash
# 解压项目文件后进入目录
cd pve-manager

# 一键启动
chmod +x start.sh
./start.sh
```

### 2. 访问应用
- 前端界面: http://localhost:5173
- 后端API: http://localhost:3000
